# Clique (vanilla 1.12.1 / Kronos)

Аддон click-casting для World of Warcraft **1.12.1** (приватный сервер
WoWCircle «Classic Kronos», Interface `11200`). Позволяет назначать заклинания
на кнопки мыши и модификаторы и применять их прямо кликом по панелям юнитов.

Оригинальный автор — **Cladhaire**. Это Ace2-версия Clique 2006 года,
использующая систему плагинов (`Plugins/`) для интеграции с конкретными
аддонами панелей юнитов.

## Локальные изменения

- **Добавлена поддержка [LunaUnitFrames](../LunaUnitFrames)** — плагин
  `Plugins/Clique_LunaUnitFrames.lua`. Он подключается к хуку
  `Luna_Custom_ClickFunction`, который Luna вызывает в обработчике клика каждой
  юнит-кнопки, и передаёт клик в `Clique:OnClick`. Биндинги Clique имеют
  приоритет; для непривязанных кнопок отрабатывает встроенный click-casting
  Luna. Файл зарегистрирован в `Plugins/Plugins.xml`.

## Установка

Скопировать папку `Clique` в `Interface/AddOns/` клиента 1.12.1 и включить
аддон на экране выбора персонажа.

## Использование

- `/clique` — открыть настройки и управлять биндингами.
